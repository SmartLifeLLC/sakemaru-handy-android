## 🎨 Step 1 — 헤더 디자인

> 配送コース選択 화면의 헤더 컴포넌트를 재사용하거나 동일 사양으로 적용.

```
배경         : 투명 (#FFFFFF 또는 배경색과 동일)
높이         : 56dp
하단 테두리  : 1dp solid #E0E0E0
```

### 헤더 내부 레이아웃 (좌 → 우)

```
[←]  [📦 아이콘]  [出庫]  [코스 뱃지*]  [날짜*]  [│]  [창고명*]  →  [⚙️*]

* 표시 = Step 0 에서 구현 확인된 경우에만 표시
```

| 요소 | 사양 |
|------|------|
| ← 뒤로가기 | 색상 `#C0392B` / 24dp |
| 📦 아이콘 | 색상 `#E67E22` / 22dp / 좌측 8dp 여백 |
| 「出庫」타이틀 | 18sp / Bold / `#C0392B` / 아이콘과 6dp 간격 |
| 코스명 뱃지 | 배경 `#27AE60` / 텍스트 White 13sp Bold / corner 20dp / 패딩 4dp×12dp |
| 날짜 | 테두리 2dp `#E67E22` / corner 8dp / 텍스트 `#E67E22` 13sp Bold / 패딩 4dp×10dp |
| 「│」구분선 | 색상 `#CCCCCC` / 좌우 10dp |
| 창고명 | 16sp / Regular / `#E67E22` |
| ⚙️ 설정 | 우측 끝 / 색상 `#555555` / 24dp |

## 🎨 Step 2 — 화면 레이아웃 구조

구현 확인된 항목에 따라 아래 두 가지 중 적합한 구조를 선택한다.

### 패턴 A — 이력 패널이 구현된 경우

```
┌────────────────────────────────────────────────┐
│  헤더                                          │
├───────────────────────┬────────────────────────┤
│  좌측 패널 (입력)     │  우측 패널 (이력)      │
│  weight: 1            │  weight: 1             │
│                       │                        │
└───────────────────────┴────────────────────────┘
```

### 패턴 B — 이력 패널이 미구현인 경우

```
┌────────────────────────────────────────────────┐
│  헤더                                          │
├────────────────────────────────────────────────┤
│                                                │
│  단일 패널 (입력 영역, 최대 너비 600dp 중앙)   │
│                                                │
└────────────────────────────────────────────────┘
```

## 🎨 Step 3 — 좌측 패널 (입력 영역) 디자인

패널 배경: `#FFFFFF` / 패딩: 16dp / 하단 rounded corner 있으면 12dp

### 3-1. 바코드 입력 행

```
레이아웃      : 입력 필드 + 아이콘 버튼들  가로 배치
* 필수 마크   : 「*」 색상 #E74C3C / 14sp / 입력 필드 앞

[바코드 텍스트 입력 필드]
  배경        : #FFFFFF
  테두리      : 1dp solid #CCCCCC / corner 8dp
  패딩        : 12dp
  폰트        : 16sp
  flex        : 1 (나머지 너비 전체)

[버튼들 — 구현된 것만 표시, 우측 배치]
  크기        : 44 × 44dp 각각
  corner      : 8dp
  테두리      : 1dp solid #CCCCCC
  배경        : #FFFFFF
  아이콘 색   : #555555
  버튼 간격   : 6dp
  press 시    : 배경 #F0F0F0
```

### 3-4. 로케이션 표시 (구현된 경우)

```

  폰트      : 14sp / Bold / #212529
  우측 여백 : 8dp

  배경      : #FFFDE7  (연노랑)

  corner    : 6dp
  패딩      : 10dp(수직) × 16dp(수평)
  폰트      : 16sp / Bold / #212529
  정렬      : 중앙
  flex      : 1
```

    폰트    : 14sp / #212529
