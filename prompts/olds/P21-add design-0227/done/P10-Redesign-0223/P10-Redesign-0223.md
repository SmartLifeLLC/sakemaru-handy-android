## 📂 Step 0 — 현재 화면 파악 (착수 전 필수)

```bash
# pages.md 로 파일명 확인
cat sakemaru-handy-android/pages.md

# 入庫 관련 파일 검색
find sakemaru-handy-android -type f -name "*.kt" | \
  xargs grep -l "入庫\|Inbound\|inbound\|nyuuko" 2>/dev/null

find sakemaru-handy-android -type f -name "*.xml" | \
  xargs grep -l "inbound\|nyuuko" 2>/dev/null

### 구현 여부 확인 (코드 확인 후 기재)

아래 항목을 코드에서 직접 확인한다. **구현된 항목만 UI에 반영한다.**

```
[헤더 영역]
□ 뒤로가기 버튼        → 있음 / 없음
□ 아이콘 + 「入庫」타이틀 → 있음 / 없음
□ 날짜 뱃지            → 있음 / 없음
□ 창고명               → 있음 / 없음
□ 설정 아이콘 (⚙️)    → 있음 / 없음

[좌측 패널]
□ 바코드 텍스트 입력   → 있음 / 없음
□ 검색 버튼 (🔍)       → 있음 / 없음
□ 바코드 스캔 버튼     → 있음 / 없음
□ 이미지 스캔 버튼     → 있음 / 없음
□ 입고 타입 드롭다운   → 있음 / 없음  (「フリー入庫」등)
□ 로케이션 드롭다운    → 있음 / 없음  (「未設定」×2)
□ 賞味期限 날짜 입력   → 있음 / 없음
□ 상품 정보 표시 영역  → 있음 / 없음
□ 빈 상태 안내 문구    → 있음 / 없음

[우측 패널]
□ 履歴 패널 자체       → 있음 / 없음
□ 履歴 건수 뱃지       → 있음 / 없음
□ 同期 버튼            → 있음 / 없음
□ 이력 카드 목록       → 있음 / 없음
□ 이력 카드 — 一致 뱃지 → 있음 / 없음
□ 이력 카드 — 수정 버튼 → 있음 / 없음
□ 이력 카드 — 삭제 버튼 → 있음 / 없음
```

```

헤더 배경            : #FFFFFF  (또는 화면 배경과 동일)

--- 초록 메인 색 ---
강조색 (메인 그린)   : #27AE60
강조색 press 시      : #1E8449

날짜 뱃지 테두리     : #27AE60
날짜 뱃지 텍스트     : #27AE60
同期·액션 버튼 배경  : #27AE60
同期·액션 버튼 텍스트: #FFFFFF
一致 뱃지 배경       : #27AE60
一致 뱃지 텍스트     : #FFFFFF

--- 텍스트 ---
텍스트 주색          : #212529
텍스트 보조색        : #555555

--- 카드 / 입력 ---
카드 배경            : #FFFFFF

--- 공통 ---

## 🎨 Step 1 — 헤더 디자인

```
배경색       : #FFFFFF  (또는 투명)
높이         : 56dp
하단 테두리  : 1dp solid #D5F5E3
```

### 헤더 내부 레이아웃 (좌 → 우)

```
[←]  [📦 아이콘]  [入庫]  [날짜 뱃지*]  [│]  [창고명*]  →  [⚙️*]

* 구현 확인된 경우에만 표시
```

| 요소 | 사양 |
|------|------|
| ← 뒤로가기 | 색상 `#27AE60` / 24dp / 좌측 12dp |
| 📦 아이콘 | 색상 `#27AE60` / 22dp / 좌측 8dp |
| 「入庫」타이틀 | 18sp / Bold / `#27AE60` / 아이콘과 6dp 간격 |
| 날짜 뱃지 | 테두리 2dp `#27AE60` / corner 8dp / 텍스트 `#27AE60` 13sp Bold / 패딩 4dp×10dp |
| 「│」구분선 | `#CCCCCC` / 좌우 10dp |
| 창고명 | 16sp / Regular / `#27AE60` |
| ⚙️ 설정 | 우측 끝 / `#555555` / 24dp |

## 🎨 Step 3 — 좌측 패널 디자인 (구현 항목만)

패널 배경: `#FFFFFF` / corner 12dp / 패딩 12dp / 테두리 1dp `#D5F5E3`

### 3-1. 바코드 입력 행

```
레이아웃    : [*] [입력 필드] [🔍] [바코드] [이미지]  가로 배치

* 필수 마크
  텍스트    : 「*」/ 색상 #E74C3C / 14sp / Bold

입력 필드
  배경      : #FFFFFF
  테두리    : 1dp solid #CED4DA / corner 8dp
  패딩      : 12dp
  placeholder : 「商品コード/JAN」/ 색상 #AAAAAA
  폰트      : 16sp
  flex      : 1

아이콘 버튼 (구현된 것만)
  크기      : 44×44dp / corner 8dp
  테두리    : 1dp solid #CED4DA
  배경      : #FFFFFF
  아이콘 색 : #555555
  간격      : 6dp
  press 시  : 배경 #F0F0F0
```

### 3-2. 입고 타입 드롭다운 (구현된 경우)

```
배경        : #FFFFFF
테두리      : 1dp solid #27AE60  ← 초록 테두리
corner      : 8dp
패딩        : 12dp
폰트        : 15sp / #212529
▼ 아이콘    : #27AE60
상단 여백   : 8dp
```

### 3-3. 로케이션 + 賞味期限 행 (구현된 경우)

```
레이아웃    : 「ロケ:」라벨 + [드롭다운1] - [드롭다운2]  +  「賞味期限:」+ [날짜 입력]  가로 배치

「ロケ:」라벨
  폰트      : 14sp / Bold / #212529
  우측 여백 : 6dp

로케이션 드롭다운 (각)
  배경      : #FFFFFF
  테두리    : 1dp solid #27AE60
  corner    : 8dp
  패딩      : 8dp×12dp
  폰트      : 14sp / #212529
  ▼ 아이콘  : #27AE60

「-」구분자
  폰트      : 14sp / #888888 / 좌우 4dp

「賞味期限:」라벨
  폰트      : 12sp / #555555
  좌측 여백 : 8dp

날짜 입력 필드
  배경      : #FFFFFF
  테두리    : 1dp solid #CED4DA
  corner    : 8dp
  패딩      : 8dp×12dp
  placeholder : 「yyyy/mm/dd」
  폰트      : 13sp
  📅 아이콘  : #555555
```

### 3-4. 빈 상태 영역 (구현된 경우)

상품 코드 입력 전 표시되는 안내 영역:

```
레이아웃    : 영역 중앙 배치 (수직+수평)
아이콘      : ⓘ 또는 ! / 색상 #AAAAAA / 18dp
텍스트      : 「商品コードを入力してください」
폰트        : 14sp / #888888
아이콘과 텍스트 간격 : 6dp (가로 배치)
```

## 🎨 Step 4 — 우측 패널 (이력 영역) 디자인 (구현된 경우)

### 4-1. 패널 헤더

```
배경         : #F8F8F8
패딩         : 12dp
하단 테두리  : 1dp solid #EEEEEE
레이아웃     : 좌측 [履歴 + 건수 뱃지]  |  우측 [同期 버튼*]

「履歴」
  폰트       : 15sp / Bold / #212529

건수 뱃지
  배경       : #E74C3C (빨강) / 원형 / 최소 20dp
  폰트       : 12sp / Bold / White
  위치       : 「履歴」텍스트 우측 상단

「同期」버튼 (구현된 경우)
  배경       : #27AE60 (초록)
  corner     : 6dp
  패딩       : 6dp×14dp
  폰트       : 13sp / Bold / White
  press 시   : 배경 #1E8449
```

### 4-2. 이력 카드

```
배경         : #F0FFF4  (연한 민트)
테두리       : 1dp solid #A5D6A7 (연한 초록)
corner       : 10dp
패딩         : 12dp
그림자       : elevation 1dp
카드 간격    : 8dp

[1행] 상품명 + 一致 뱃지 (우측)
  상품명     : 15sp / Bold / #212529
  flex       : 1

[2행] 바코드 / 규격 / 로케이션
  폰트       : 12sp / #555555
  로케이션   : 색상 #1A7A4A (짙은 초록)

[3행] 入庫 / 発注 건수
  폰트       : 12sp / #555555

[우측 액션 버튼들 — 구현된 것만]
  수정       : 테두리 1dp #CCCCCC / corner 6dp / 아이콘 #555555 / 36×36dp
  삭제       : 테두리 1dp #FFCCCC / corner 6dp / 아이콘 #E74C3C / 36×36dp
  간격       : 6dp
```

### 4-3. 一致 뱃지 (구현된 경우)
